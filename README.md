# dasrobot_pkgs

Набор ROS‑пакетов для управления устройством в рамках экосистемы DASRobot.  
Репозиторий объединяет несколько ROS‑пакетов для унифицированной сборки и развертывания.

## Состав

В репозитории находятся следующие пакеты:

- `dasrobot_description` — описание робота
- `dasrobot_navigation` — навигация
- `dasrobot_simulation` — симуляция робота в среде gazebo
- `dasrobot_voice` — распознавание и генерация голоса

## Установка и сборка

1. Клонируйте репозиторий в `src/` вашего workspace:

   ```bash
   cd ~/ros2_ws/src
   git clone https://github.com/DASRobotDevelopment/dasrobot_pkgs.git
   ```

2. Соберите workspace:

   ```bash
   cd ~/ros2_ws
   colcon build
   ```

3. Активируйте окружение:

   ```bash
   source install/setup.bash
   ```
